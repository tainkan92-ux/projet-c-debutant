# Projets en langage C

Ce dépôt contient mes premiers projets réalisés en langage C dans le cadre de mon apprentissage en autonomie depuis octobre 2025.

## Projets

### Calculatrice
Programme permettant d’effectuer des opérations simples (addition, soustraction, multiplication, division).

### Quiz interactif
Programme posant plusieurs questions à l’utilisateur avec un score final.

### Programme de gestion bancaire
Programme permettant de :
- consulter un solde
- déposer de l’argent
- retirer de l’argent (avec vérification du solde)

## Objectif

Ces projets m’ont permis de travailler :
- les conditions
- les boucles
- les fonctions
- la logique algorithmique

Je continue actuellement à progresser en programmation, notamment en Python.
